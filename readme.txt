===== SETUP =====

The web site is located at kennyschank.com/demo/dnd-tools. The Player Inventory tool is located at kennyschank.com/demo/dnd-tools/dnd.php.

dnd.sql contains all the MySQL queries needed to get the database and its tables up and running.

===== USAGE =====

You can access the Player Inventory by clicking on Tools > Player Inventory.


In that tool:

• To filter, click the funnel icon in the top right corner.
• To sort the table of owners, click the column you wish to sort.
