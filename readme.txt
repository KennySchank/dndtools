===== SETUP =====


The web site is located at hpd3/.

dnd.js and dnd.css need to be stored in the same directory as dnd.php.

dnd.sql contains all the MySQL queries needed to get the database and its tables up and running.




===== USAGE =====


You can access the Player Inventory by clicking on Tools > Player Inventory.


In that tool:

• To filter, click the funnel icon in the top right corner.

• To sort the table of owners, click the column you wish to sort.



===== OTHER INFORMATION =====


total_weight_sql_query.txt contains an SQL query that will return the total weight of all owner inventories.


REST_endpoints.xlsx contains a mapping of the REST endpoints needed for this website.
